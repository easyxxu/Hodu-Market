# Hodu Market

호두마켓은 판매자와 구매자를 구별하여 상품 등록, 수정, 구매 기능을 제공하는 오픈마켓 서비스입니다.

**배포 사이트** : https://effervescent-liger-740f87.netlify.app/

## **테스트 아이디 & 비밀번호**

| 로그인 유형 | 아이디       | 비밀번호   |
| ----------- | ------------ | ---------- |
| **구매자**  | `hoduubuyer` | `123123as` |
| **판매자**  | `hoduseller` | `123123qw` |
